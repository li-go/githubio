### Install

```
$ go get -u github.com/li-go/goserve
```

### Usage

```
$ goserve -h
Usage of goserve:
  -dir string
        directory to host (default ".")
  -disable-cache
        disable http cache
  -port int
        port to host (default 3000)

```
